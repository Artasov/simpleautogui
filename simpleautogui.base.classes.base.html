findText(text: str, lang: str = 'eng+rus', contrast: int | float = 0, resize: int = 0, sharpen: bool = True, case_sensitive: bool = False, min_confidence: int = 80, **image_to_data_kwargs) → list[Region][source]#

Searches for the specified text within the region using OCR and returns regions containing the text.

Parameters:

  • text – The text to search for.

  • lang – Language(s) for OCR, separated by a plus sign (e.g., ‘eng+rus’). Default is ‘eng+rus’.

  • contrast – Level of contrast enhancement to apply to the image. 0 means no enhancement. Can be a float.

  • resize – Scaling factor to apply to the image. 0 means no scaling. This is an integer value.

  • sharpen – Whether to apply a sharpening filter to the image. Default is True.

  • case_sensitive – Whether the search should be case-sensitive. Default is False.

  • min_confidence – The minimum confidence level (from 0 to 100) to accept a text match. Default is 0.

  • image_to_data_kwargs – Additional keyword arguments for pytesseract.image_to_data.

Returns:

A list of Region objects containing the matched text. Each region represents the bounding box of the text.
